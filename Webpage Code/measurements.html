OBD2 Vehicle Scanner


        
    


    


    






    
Instructions

    
    
    
  • Module Selection:
        
      
            
    • Type ECM, TCM, or ABS to connect to the respective module (case-insensitive).
      • 
        
    
    
    • 

    
  • Data Collection:
        
      
            
    • Click 'start' to begin reading live data from the connected module.
      • 
            
    • Click 'stop' to stop reading live data.
      • 
        
    
    
    • 

    
  • Diagnostic Trouble Codes (DTCs):
        
      
            
    • Type 'dtc' to read available Diagnostic Trouble Codes (DTCs).
      • 
            
    • After reading DTCs, you can select the Preferred Mode to filter your results:
                
        
                    
      • '03' for Mode 03 (Stored DTCs).
        • 
                    
      • '07' for Mode 07 (Pending DTCs).
        • 
                
      
            
      • 
        
    
    
    • 

    
  • Clear DTCs and Disable MIL:
        
      
            
    • Type 'clear' to clear the DTCs and disable the MIL (Malfunction Indicator Light).
      • 
        
    
    
    •
